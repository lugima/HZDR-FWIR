1) Execute sort_mpa.exe. This will show a message in the console.
2) Copy the directory of the folder with the .mpa files you want to sort and paste it into the console.
   TIP: To copy the route of the folder, open it and right-click its name in the File Explorer upper bar.
        Then click "Copy address as text".
3) The .xlsx files will be created in the .exe location with the names "[.mpa filename].xslx".
